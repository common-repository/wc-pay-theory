A Plugin to accept payments with Pay Theory through Woocommerce.

== Description ==

Accept Visa, MasterCard, American Express, Discover, JCB, Diners Club payments in your WooCommerce store with the Pay Theory's payment gateway for WooCommerce.

= Automatic Installation =

Automatic installation is a convenient method since WordPress manages file transfers and you don't need to leave your browser. To automatically install the WooCommerce Pay Theory plugin, follow these steps:

1. Log in to your WordPress dashboard.
2. Make sure you have installed and activated WooCommerce.
3. Go to the Plugins menu and click "Add New".
4. Search for "WC Pay Theory" and find our plugin.
5. Click "Install Now" and then "Activate" to complete the installation.

= Manual installation =

To manually install our plugin, download it and upload to your web server using an FTP application. Follow the [WordPress codex guide](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation) for detailed instructions.

= Updating =

Automatic updates are generally reliable, but it's important to verify that your store's plugins are up-to-date by going to the "Plugins" tab on your administration page.

= Quick Start =

1. Go to your administration page and click "Plugins" in the left-hand menu.
2. Click "Add New" and search for "WC Pay Theory".
3. Click "Install Now" and then "Activate" to complete the installation.
4. Go to the WooCommerce settings page and click the "Settings" then the "Payments" tab.
5. Click "Manage" next to Pay Theory.
6. Check boxes labeled: "Enable Payments" and "Enable Test Mode"
7. Enter your Pay Theory Test API Key and click "Save Changes".
8. Checkout with Pay Theory!

== Frequently Asked Questions ==

= How do I enable test mode? =

Go to the WooCommerce settings page and click the "Settings" then the "Payments" tab. Click "Manage" next to Pay Theory. Then click the checkbox "Enable Test Mode" and save changes.

= How do I enable transaction email reciepts? =

Go to the WooCommerce settings page and click the "Settings" then the "Payments" tab. Click "Manage" next to Pay Theory. Then click the checkbox "Enable Email Receipts" and save changes.

= How do I enable production mode? =

Go to the WooCommerce settings page and click the "Settings" then the "Payments" tab. Click "Manage" next to Pay Theory. Then ensure the checkbox "Enable Test Mode" is not checked. Ensure you have a valid production API key entered in the "Production/Live API Key" field.
